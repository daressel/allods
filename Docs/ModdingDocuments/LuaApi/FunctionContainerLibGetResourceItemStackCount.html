На главную

containerLib.GetResourceItemStackCount( slotType, id )

Возвращает количество предметов в контейнере с тем же ресурсом что и у данного.

Типы контейнеров: ITEM_CONT_....

-- библиотека:
containerLib

-- объявление:
GetResourceItemStackCount( slotType, id )

-- параметры:
slotType: number (enum ITEM_CONT_...) - тип контейнера с предметами
id: number (ObjectId) or ItemId (ResourceId) - идентификатор предмета


-- возвращаемые значения:
number - количество предметов

Пример:

local count = containerLib.GetResourceItemStackCount( ITEM_CONT_INVENTORY, itemId )
