На главную

containerLib.IsItemSlotEnabled( slotType, slot )

Проверяет наличие активных транзакций связанных с указанным слотом указанного контейнера. При наличии таковых выполнение любых действий (исключая получение информации) со слотом запрещено.

Слот должен быть валидным. Если такого слота в контейнере не существует - будет выдана ошибка.

См. ITEM_CONT_....

-- библиотека:
containerLib

-- объявление:
IsItemSlotEnabled( slotType, slot )

-- параметры:
slotType: number (enum ITEM_CONT_...) - тип контейнера с предметами
slot: number - номер слота в контейнере


-- возвращаемые значения:
boolean: true - если активных транзакций с данным слотом нет и можно выполнять операции, в противном случае false

Пример:

local actionsAvailable = containerLib.IsItemSlotEnabled( ITEM_CONT_INVENTORY, 12 )
