enum DUMMY_TYPE

Тип контейнера с предметами экипированными на персонаже игрока.

Перечисление определено в коде, значения:

DUMMY_TYPE_PRIMARY - первичный набор экипировки
DUMMY_TYPE_SECONDARY - вторичный набор экипировки (первый набор экипировки для "драконьего облика")
DUMMY_TYPE_TERTIARY - третичный набор экипировки (второй набор экипировки для "драконьего облика")
DUMMY_TYPE_QUATERNARY - четвертичный набор экипировки (третий набор экипировки для "драконьего облика")
