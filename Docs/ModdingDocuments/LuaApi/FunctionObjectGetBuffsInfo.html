На главную

object.GetBuffsInfo( data, includeResourceInfo, ignoreInvisibleBuffs )

Возвращает таблицу hash_map<buffId, buffInfo>. Ключи - идентификаторы экземпляров бафа, значения - таблицы информации о бафах. Формат таблиц идентичен object.GetBuffInfo

-- система:
UIState/GameState

-- библиотека:
object

-- объявление:
function GetBuffsInfo( Id )

-- параметры:
data: 2 варианта
  ObjectId - идентификатор интерактивного объекта для которого нужно получить информацию о бафах
  table[1] of ObjectId - таблица (индексированная с 1) идентификаторов экземпляров бафов для которых нужно получить информацию
includeResourceInfo: boolean or nil - Включать ли информацию о ресурсе бафа при получении информации о экземпляре бафа. По умолчанию true
ignoreInvisible: boolean or nil - игнорировать бафы с needVisualize == false. По умолчанию false.

-- возвращаемые значения:
table (hash_map<buffId,  buffInfo>). Ключи - идентификаторы экземпляров бафа, значения - таблицы информации о бафах.

-- пример:
LogInfo( "Список видимых бафов на аватаре:" )
for buffId, buffInfo in pairs( object.GetBuffsInfo( avatar.GetId(), true, true ) ) do
  LogInfo( buffId, buffInfo.name )
end
