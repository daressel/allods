EVENT_EQUIPMENT_DRESS_ENABLE_CHANGED

Событие посылается, когда главному или инспектируемому игроку становятся доступны ритуальные вещи (контейнер, действия и т.п.).

-- поля
unitId: ObjectId - идентификатор игрока, у которого изменилось состояние
type: number(enum DUMMY_TYPE) - тип контейнера с экипировкой

Список контейнеров с экипировкой см. в enum DUMMY_TYPE.
